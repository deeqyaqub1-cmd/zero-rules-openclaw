How it works

Three steps. No config.

01

Intercept

ZeroRules checks every query against 5 rule patterns before the LLM sees it. Math, time, currency, files, dates.

02

Resolve locally

Matched queries execute in <5ms with zero tokens. Calculator for math. IANA database for time. API for currency.

03

Pass through

No match? ZeroRules steps aside. Your query goes to the LLM exactly as before. Zero interference with creative tasks.
